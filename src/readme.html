💮 SmartCalc v2.0 документация: 💮

SmartCalc v2.0 приложение для удобных вычислений и графиков.

Установка и запуск:

  • Для простой установки предусмотрен "мэйкфайл" с целью install
  • Для запуска есть цель run (по умолчанию цель run предусматривает выполнение цели install)
  • Саму программу можно найти в каталоге "src/app/"
  • Для корректной установки потребуется qmake6
  • Если возникают ошибки во время цели style скорее всего у вас не установлен cppcheck установите или закоментируйте первую сьроку в цели style

Интерфейс программы:

  • Программа представленна ввиде простого строкового калькулятора, так же предусмотрена отрисовка графиков. Удобство интерфейса заключается в том что мышка компьютера используется по минимуму, а с помощью клавиатуры не нужно набирать все функции целикм, достаточно нажать одну клавишу с которой начинается имя функции, ниже представлена карта для удобства. Между полями можно переключаться с помощью клавиши TAB.

  • Для набора выражения фокус должен быть сосредоточен на главном поле для ввода, у каждого поля свое предназначение лишнее программа не позволит вам ввести дабы снизить ошибки во время работы, вам лишь остается правильно ввести выражение и нажать кнопку Equal или кнопку Print Graph для построения графика.

  • Клавиша Backspace стирает посимвольно, в то время как клавиша Delete очищает окно ввода.

  • Карта сокращений:
    • c
      • - cos()
    • C
      • - acos()
    • s
      • - sin()
    • S
      • - asin
    • t
      • - tan
    • T
      • - atan
    • l
      • - ln()
    • L
      • - log()
    • q
      • - sqrt()
    • ^
      • - pow()
    • %
      • - mod()
    • x
      • - x
    • e
      • - для записаи длинных чисел в научной форме.
  • P.S: Потдерживается Drag and drop, просто перетащите выражение в поле ввода.

    • Makefile:

    • install
      • Выполнит установку программы

    • run
      • Установит и запустит программу (не обязательно ее прописывать можно просто напечатать make)

    • uninstall
      • Удалит программу

    • tests
      • Протестирует функции написанные на си++

    • dvi
      • Откроет данную документацию

    • gcov_report
      • Соберет статистику по модульному тестированию и откроет в браузере результат

    • dist
      • Выполнит создание tar-файла содержащего полный установочный пакет программы SmartCalc v2.0

    • clean
      • Удалит все ненужные и временные файлы

    • style
      • Проверит написанную библиотеку на google стиль
